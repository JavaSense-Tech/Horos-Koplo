# Horos-Koplo GitHub Pages

Static site package for the Horos-Koplo Android app.

## Included pages
- index.html
- privacy.html
- terms.html
- support.html
- delete-data.html

## Current app assumptions
This package is written for the current Horos-Koplo app state discussed in development:
- no mandatory user account
- core reading/profile data stored locally on device
- support contact via email

If the app later adds sign-in, analytics, advertising, cloud sync, crash reporting, or any server-side account storage, update these pages before release.

## How to publish on GitHub Pages
1. Create a public GitHub repository.
2. Upload all files from this folder to the repository root.
3. In GitHub repo settings, open **Pages**.
4. Choose the main branch as the publishing source and save.
5. Use the published URL:
   - `/privacy.html` for the Play Console Privacy Policy field
   - `/delete-data.html` as the web page for deletion information if needed
