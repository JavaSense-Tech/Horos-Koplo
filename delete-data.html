Delete Data

How user data is deleted in the current version of Horos-Koplo
Horos-Koplo currently does not require users to create an in-app account for the core reading experience.

Local app data

In the current version of Horos-Koplo, core reading/profile data is primarily stored locally on your device. You can generally remove it by:

  • Using any available in-app reset option, if present
  • Clearing app data in your Android device settings
  • Uninstalling the app

Email support records

If you have emailed support and want support correspondence removed where reasonably possible, send a request to radenraradewianggraini@gmail.com with the subject line Horos-Koplo Data Deletion Request.

If future versions add accounts

If a future version of Horos-Koplo adds in-app account creation, this page will be updated to provide the required web-accessible deletion method and instructions.
